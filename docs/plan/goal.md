# プロジェクトの目的・完成形

## 目的

ノードグラフ上で Stable Diffusion 用プロンプトを「意味単位」で組み立て、組み合わせ（直積）で複数枚を一括生成するデスクトップアプリを作る。詳細仕様は [../spec.md](../spec.md)。

## 完成形

- キャラ・背景・アクション・カメラなどを意味ノードとして編集できる。
- ノードの束ね/分けで「連動 vs 直積」を制御し、組み合わせ生成のスイープ軸を設計できる。
- Scene ノードが上流をたどってプロンプトにコンパイルする。
- 既存生成画像（メタデータ付き）を参照ノードとして読み込み、キャラだけ/背景だけ差し替えたバージョンを生成できる。
- 描画は WebUI Forge（A1111 互換 API）。領域分割系は使わない。
- 可視性フィルタ・参照分解にローカル LLM（llama.cpp）を使う。

## 重視する価値

- **再現性**: 「あの設定でもう一度」を成立させる。LLM 判定はキャッシュ＋手修正可能に。
- **直積の安全性**: ドライランで総枚数・サンプルを提示してから確定（爆発の安全弁）。
- **構造で意図を表す**: 鎖・ハンドル型・ノード境界で束縛を明示し、挿し間違いを防ぐ。

## 技術方針（確定）

- TypeScript 一本（Electron main で完結）。Python/FastAPI は使わない。lm-graph を参照に使える部分を流用する。
